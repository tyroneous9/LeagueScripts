1. Save "resources" folder and "config.cfg" elsewhere. You will need these every update.

2. If this is the first time setup, change your settings using the "Change-Settings" script (make sure to hit save). Also, replace the images in "resources" with your own.

3. Keybindings when running a Bot:
	Home		|	Starts the bot
	End		|	Pauses the bot
	Delete		|	Exits the bot
	Insert		|	Runs the test function (edit inside the bot's script)
